# Dynamic Validation Evidence

This GitHub package keeps the compact dynamic validation reports and machine-readable summaries:

- `dynamic_validation_report.md`
- `dynamic_validation_summary.json`
- `repeated_dynamic_validation_report.md`
- `repeated_dynamic_validation_summary.json`

Large raw kGym job contexts, kernel configs, VM logs, and intermediate diagnostic attempts are intentionally excluded from the GitHub-ready folder. The project report records the important result boundary: dynamic validation covers only the representative `bug_008 improved` case, not all 24 model outputs.
